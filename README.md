# SuperStore Sales Power BI Dashboard

## Project Overview

This Power BI dashboard analyzes SuperStore sales data and provides business insights through interactive visualizations.

## Dashboard Pages

- Executive Summary
- Product Analysis
- Regional Analysis
- Customer Analysis
- Time Analysis
- Business Insights

## Key KPIs

- Total Sales
- Total Profit
- Total Orders
- Total Customers
- Total Quantity
- Profit Margin %

## Tools Used

- Power BI
- DAX
- Power Query
- Data Modeling

## Project Highlights

- Interactive navigation buttons
- Dynamic KPI cards
- Regional performance analysis
- Customer segmentation
- Time-based trend analysis
- Business insights page
